# Place your CODESYS Docker images here

Download CODESYS Control for Linux ARM SL Docker images from:
- CODESYS Store: https://store.codesys.com
- GitHub Releases (if available)

Expected files:
- `codesys-arm64.tar` (for ARM64/ARMv8 devices)
- `codesys-arm32.tar` (for ARM32/ARMv7 devices)

These files are excluded from git via .gitignore due to their size.
